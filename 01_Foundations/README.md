# HEP Analysis Foundations

Core module for environment setup and initial exploration of high-energy physics data.

## Directory Structure

* **`01_HEP_Analysis_Stack/`**: Contains sub-modules for `Array_Programming` (NumPy/Awkward) and `Machine_Learning`.
* **`02_Containerization/`**: Docker configurations to ensure a reproducible execution environment.
* **`initial_inspection.py`**: A lightweight script to validate ROOT files and list their branches (TTrees).
